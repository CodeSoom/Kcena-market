## 기술 스택
- JavaScript
- React
- Redux, Redux Toolkit
- Emotion(styled)
- Firebase

## Install dependencies
```
npm install
```

## Lint
```
npm run lint
```

## Start dev server
```
npm run start
```

## Run build
```
npm run build
```

## Run coverage test
```
npm run coverage
```

## Run unit test
```
npm run test:unit
```
